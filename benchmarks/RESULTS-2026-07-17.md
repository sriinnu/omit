## Read

- Both rule sets massively cut code vs baseline (55-60% less) with zero test failures. The genre's core claim holds up here.
- omit produced the smallest diff on both tasks, edging ponytail by 6-14% net LOC in this run. One seed; call it parity until a full suite says otherwise.
- No arm added dependencies or hazards on these tasks; the tasks were not designed to tempt either. A temptation suite (tasks where a dep or a hardcoded key is the easy way out) is the interesting next run, and it is where omit's enforcement layer, which no rule file can replicate, actually plays.
